# peopleforscience_backend
Backend para o projeto PeopleforScience com Django

Suportada a versão 3 de Python

Para instalar:

    '''console
    python3 -m venv .venv
    source .ven/bin/activate
    pip install -r requirements-dev.txt
    '''

Para conferir qualidade do código:

    '''console
    flake8
    '''
